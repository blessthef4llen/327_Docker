# CECS 327 - Introduction to Networks and Distributed Computing

A project repository for CSULB's CECS 327 - Introduction to Networks and Distributed 
Computing. Each project folder contains implementations of core networking and distributed 
systems concepts covered in the course.

## Repository Structure
- `proj1-docker-containers/` - Docker containerization fundamentals
  - `python_app/` - Custom Python Docker image and Nginx web server
  - `server-clients/` - Multi-container TCP echo server with multiple clients
- `proj2-distributed-communication/` - Distributed communication patterns
  - `task1-anycast-with-docker/` - Simulating Anycast routing with Docker and TCP
  - `task2-multicast-with-udp/` - IP Multicast broadcasting with Docker and UDP
